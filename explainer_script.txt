IntelliLead Demo Video Script (Detailed, ≈3–4 minutes)

Goal: Record a walkthrough exactly in this order on http://localhost:8502/ :
1) Dashboard tab  → 2) Leads tab  → 3) Chat tab

====================================================
0:00–0:20 — Cold open
“Hi everyone. Today I’m showing IntelliLead—an AI lead intelligence demo built with Streamlit and SQLite.
What makes it useful is that it doesn’t require any paid AI API key for this demo.
Instead, it captures leads, stores them, runs an automation workflow, and generates lead-aware responses using a lightweight offline keyword-based AI.”

(As you speak, show the home page.)

====================================================
0:20–1:25 — Dashboard tab walkthrough (first screen)
“First, we’re on the Dashboard.
This page is like the admin overview.

1) KPI cards:
- Total Leads: shows how many leads are stored in the system.
- Automated: how many leads are already marked completed.
- Pending: leads waiting for automation.
- AI Replies sent: how many leads have already received an AI reply.

2) Trends chart:
Next, we have ‘Leads per day’ for the last 7 days.
This helps you see whether submissions are increasing or slow.

3) Department breakdown chart:
Finally, the donut chart shows how many leads came from each program interest—like AI/Automation, Data Science, Full Stack, Cloud Ops, and Other.”

(Optional line if there’s no data yet):
“If you’re starting fresh and there’s no data yet, the charts will show empty states until you submit leads.”

====================================================
1:25–2:35 — Switch to Leads tab + show Lead Capture form
“Now let’s click the ‘Leads’ tab.

Here we have the Lead Capture form.
I’ll quickly demonstrate by entering a sample lead.

- Full Name: type any name
- Email: enter a valid email (make sure it’s not already in the database)
- Program Interest: choose one track, for example Data Science
- Message: add a quick question like ‘I want to know the course duration’
- Urgency: choose Just browsing / Interested / Ready to enroll

Then click ‘Submit Lead’.”

====================================================
2:35–3:20 — Show automation workflow animation + explain what happens
“Immediately after clicking submit, the automation workflow starts.
You can see a step-by-step status indicator:

Step 1: ‘Lead saved to database’
The lead is inserted into SQLite—so we have a persistent record.

Step 2: ‘AI reply generated’
Now the app generates an assistant reply.
Because this demo is offline-safe, it uses a keyword-based responder to produce helpful text.
It’s designed to answer admissions, pricing, course structure, duration, batch timings, placement support, certification, and refund policy.

Step 3: ‘Workflow completed’
Finally, the lead is marked as completed, and the reply is stored back into SQLite.”

(Optional realism line):
“If you look at the console output in the terminal, you can also see that it simulates sending an email confirmation.”

====================================================
3:20–3:55 — Show Admin panel editing (same Leads tab)
“Next, I’ll scroll slightly to the Admin panel section on this same tab.

This table shows all leads.
You can filter by automation status or search by department.
Then you can edit:
- automation_status
- priority

This demonstrates how the system supports operational review after the automation runs.”

====================================================
3:55–4:40 — Switch to Chat tab + demo chat behavior
“Now let’s open the ‘Chat’ tab.

In chat, you can ask business or course-related questions.
For example, I’ll type:
‘What is the duration and batch timing for Data Science?’

The assistant replies instantly using the same offline keyword responder.

Also, when a lead context exists, the app updates the lead record by incrementing the AI replies count and saving the latest assistant reply.”

(While saying this, type 1 question and show the reply.)

====================================================
4:40–4:55 — Wrap up
“So in summary, IntelliLead gives you:
- a lead capture form
- SQLite persistence
- an automation workflow
- dashboard analytics
- and a lead-aware chat assistant

All in a lightweight Streamlit app—perfect for demos, prototypes, and interview showcases.”

(Optional closing line):
“Thank you—this is IntelliLead.”
